# 📊 A/B Testing Product Analytics Dashboard

This project presents an end-to-end product analytics solution using Power BI to compare the performance of two products — **Dell Latitude 5440** and **Dell Latitude 7440** — through A/B testing.

---

## 🎯 Objective

The goal of this project is to:

- Compare product performance using A/B testing
- Analyze user behavior and engagement
- Identify funnel drop-offs
- Evaluate conversion efficiency
- Provide actionable business insights

---

## 📂 Dataset Overview

The dataset simulates user interactions across multiple tables:

- **Users** → demographic and segmentation data  
- **Sessions** → session duration and activity  
- **Events** → user journey (page view, product view, cart, checkout, purchase)  
- **Orders** → revenue and transaction data  
- **Products** → product attributes  

---

## 📈 Key Metrics

- **Product Conversion Rate** = Purchase Sessions / Product View Sessions  
- **Engagement Depth** = Events per Session  
- **Average Session Duration**  
- **Drop-off Rate**  
- **Revenue & Profit**

---

## 🔍 Key Insights

- Latitude 5440 drives higher traffic but lower conversion  
- Latitude 7440 delivers stronger conversion and revenue  
- Checkout stage shows the highest drop-off (~40%)  
- Power users contribute the majority of purchases  
- Mobile devices dominate user interactions  

---

## 💡 Business Recommendations

- Optimize checkout experience to reduce drop-offs  
- Improve conversion strategy for 5440  
- Leverage high-performing patterns from 7440  
- Target high-engagement users with personalized strategies
